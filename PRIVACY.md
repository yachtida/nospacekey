# プライバシー / Privacy

> **要約: 本ソフトウェアは、既定で入力内容を一切外部に送信しません。**
> 変換・学習・ログの全てがあなたの PC の中で完結します。
>
> *Summary: By default, this software never sends your input anywhere.
> All conversion, learning, and logging happen entirely on your PC.*

## 外部通信

- かな漢字変換（古典変換・Zenzai ニューラル変換）は**全てローカルで実行**されます。
  ニューラル変換もローカル LLM（llama.cpp / CPU）であり、クラウド変換 API は存在しません。
- ネットワークアクセスが発生するのは **Zenzai モデル（GGUF）のダウンロードのみ**です
  （インストーラの選択式タスク、または設定アプリからの明示的なダウンロード操作。取得先は
  Hugging Face の公開モデル）。ダウンロードを行わなくても古典変換で全機能が動作します。
- 自動アップデート・テレメトリ・クラッシュレポート・使用統計の送信は**ありません**。

## 学習データ

- 変換履歴からの学習データは `%LOCALAPPDATA%\nospacekey\memory`（ユーザー辞書は
  `%LOCALAPPDATA%\nospacekey\`）に**ローカル保存**され、外部送信されません。
- **パスワード入力欄では変換・学習を行いません。** パスワード欄（TSF の `IS_PASSWORD` /
  Chromium 系のパスワード専用コンテキスト）を検出すると直接入力モードに切り替わり、
  キー入力は合成・記録を経由せずそのままアプリケーションへ渡されます。
- 学習は設定アプリからオフにでき、蓄積済みの学習データもワンクリックで消去できます。

## 診断ログ

- 診断ログは**既定で無効**です。環境変数 `NOSPACEKEY_LOG=1` を設定した場合のみ
  `%TEMP%` 配下にローカル出力されます。自動で外部送信されることはありません
  （不具合報告に添付するかどうかはあなたが決めます）。

## 設定データ

- 設定は DPAPI（Windows のユーザー単位暗号化）で保護してローカル保存されます。
