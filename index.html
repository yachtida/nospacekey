nospacekey は Windows 用のかな漢字変換 IME。打つだけで変換が追いかけてくる ライブ変換で、「文節を区切って、変換して、確定して」というリズム そのものをなくします。変換も学習もすべてこの PC の中で完結。オープンソース(MIT)。

⬇ ダウンロード(Windows 11 x64) For Developers →

現在の版は開発用自己署名です。実行時に SmartScreen の警告が出た場合は「詳細情報」→「実行」で続行できます。SHA-256 は各リリースの SHA256SUMS.txt で検証できます。

CANDIDATES

できること

1
ライブ変換
打鍵に追随して自動で変換が進む。Space も Enter も、打ちたいときだけ。従来の Space 変換もそのまま使える。
2
一時日本語モード
F8(変更可)で日本語モードへ、確定した瞬間に半角英数へ自動復帰。IME の切り忘れが起きない。
ephemeral kana F8→あ メモ→確定→A
3
モードレス再変換
半角英数のまま打って、あとから選んで変換キー。モード切替を意識しない入力へ。
reconvert kaigi + 変換→会議
4
修正変換(Tab)
打ち間違えた読みを Tab 一発で修復候補に。確定すると誤読みも学習し、次からは Space 変換だけで直る。
typo repair こんんいちは + Tab→こんにちは
5
Zenzai ニューラル変換(opt-in)
GGUF モデルを置くと自動で有効化。文脈を読む変換をローカル CPU で。無ければ古典 LOUDS 変換で軽快に動く。
数字キーで候補を選ぶように、どれからでも。

SETUP

3 分ではじめる

  1. ダウンロード

    Releases から nospacekey-setup-<version>.exe を取得。

  2. インストール

    exe を実行(管理者昇格が必要 — IME の登録はマシン単位のため)。SmartScreen が出たら「詳細情報」→「実行」。

  3. Win + Space で「nospacekey」を選ぶ

    あとは打つだけ。Zenzai を使う場合は設定画面からモデルをダウンロード。

$ 煩わしいモード切替からは、おさらば。

you@win:~$ demo --ephemeral-kana # 一時日本語モード(F8)

tty1 — nospacekey IMEA

[A]=半角英数 [あ]=一時日本語モード。F8 で入り、確定した瞬間に自動で A へ戻る — 切り忘れという状態が存在しない。トリガは設定でリバインド可。

you@win:~$ man japanese-ime # 日本語 IME がつらい理由

$ ｇｉｔ　ｓｔａｔｕｓ
command not found

切り忘れが起きない。普段は半角英数のまま、日本語は F8 の一時モードで打った分だけ。確定と同時に英数へ自動復帰する。

error: expected ';' — 全角の「；」がコードに混入。全角スペースは diff でも見えない

そもそも普段が半角英数だから紛れ込む隙がない。日本語モード中ですら記号は半角のまま入る(全角にしたければ設定で)。

会議の議事録係。変換と確定の合いの手で手が止まり、話に置いていかれる

ライブ変換で打ちっぱなし — 候補が出るだけでなく確定まで自動で進むから、Space/Enter の合いの手なしで話速に食らいつける。切り忘れゼロの一時日本語モードとこの組み合わせは、他にない。

you@win:~$ cat keymap.json # 全部リバインド可(設定GUI)

{ "ephemeral": "F8", // 一時日本語モード "reconvert": "Henkan", // モードレス再変換 "mode_toggle": "Muhenkan", // ひらがな ⇄ 半角英数 "typo_repair": "Tab", // 修正変換 "commit_undo": "Ctrl+Backspace" } // ほか計12機能。衝突は保存時に検出。起動時から半角英数で始める設定もある

you@win:~$ ./get-nospacekey
